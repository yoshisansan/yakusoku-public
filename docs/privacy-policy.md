# プライバシーポリシー

吉村 明史（以下「当方」といいます。）は、当方が提供するサービス「YAKUSOKU」（以下「本サービス」といいます。）における利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

## 1. 適用範囲

本ポリシーは、本サービスにおいて当方が取得・利用する情報の取扱いに適用されます。
本サービス以外のウェブサイトやアプリケーション等における情報の取扱いについては適用されません。

## 2. 取得する情報

当方は、本サービスの提供にあたり、主に以下の情報を取得・利用します。

1. **LINEに関する情報**
   - LINEユーザーID
   - 本サービスから送信する通知メッセージの内容
     （例：Notionのタスク名・締切日・リスケ日などを含むテキスト）

2. **Notionに関する情報**
   - 本サービスで通知対象とするタスクデータベース内の情報  
     - タスク名  
     - 締切日・リスケ日  
     - その他、利用者が通知内容に含める設定とした項目（ステータスなど）

3. **技術情報等**
   - 実行日時・エラーログ等、本サービスの動作状況に関する情報
   - GitHub Actions 等の実行環境が自動的に取得するログ情報

当方は、本サービスの提供に必要な範囲に限定して情報を取得します。

## 3. 利用目的

当方は、取得した情報を以下の目的のために利用します。

1. Notionのタスク情報を取得し、締切日・リスケ日に基づいてLINEに通知するため  
2. 本サービスの安定運用・障害対応・不具合の調査のため  
3. 本サービスの改善・機能追加等の検討のため  
4. 利用規約違反行為への対応や、不正利用防止のため  
5. 法令に基づく対応のため

これらの目的の範囲を超えて利用する場合は、あらかじめ利用者の同意を得るものとします。

## 4. 個人情報の第三者提供

当方は、次のいずれかに該当する場合を除き、利用者の情報を第三者に提供しません。

1. 利用者の同意がある場合  
2. 法令に基づく場合  
3. 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難な場合  
4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要があり、本人の同意を得ることが困難な場合  
5. 国の機関もしくは地方公共団体等が法令の定める事務を遂行することに対して協力する必要がある場合

## 5. 業務委託・外部サービスの利用

当方は、本サービスの提供・運用のため、以下の外部サービスを利用することがあります。

- **LINEプラットフォーム**（LINE株式会社）  
  - LINE Messaging API を利用したメッセージ配信のため
- **Notion**（Notion Labs, Inc.）  
  - 利用者が作成したタスクデータベースへのアクセスのため
- **GitHub / GitHub Actions**（GitHub, Inc.）  
  - 本サービスのコード管理および定期実行処理のため

これらの外部サービスに対しては、本サービスの提供に必要な範囲でのみ情報が送信されます。  
各サービスのプライバシーポリシーについては、各事業者のウェブサイトをご確認ください。

## 6. 情報の保存期間

当方は、取得した情報を、利用目的の達成に必要な範囲においてのみ保存し、目的達成後は適切な方法で削除または匿名化します。  
ただし、法令により一定期間の保存が義務付けられている情報については、この限りではありません。

## 7. 安全管理措置

当方は、利用者の情報について、漏えい・滅失・き損等を防止するため、必要かつ適切な安全管理措置を講じます。  
また、これらの措置を継続的に見直し、改善に努めます。

利用者は、当方が保有する自己の個人情報について、開示・訂正・追加・削除・利用停止等を求めることができます。

なお、ご本人確認のための手続きや、法令上応じられない場合があることを、あらかじめご了承ください。

## 9. 未成年の利用者について

未成年の方が本サービスを利用する場合は、保護者の同意を得たうえでご利用ください。
